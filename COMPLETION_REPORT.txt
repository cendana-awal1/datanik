================================================================================
   DATABASE SCHEMA
================================================================================

TABLE: customers
├── id UUID PRIMARY KEY (auto-generated)
├── nik VARCHAR(50) NOT NULL UNIQUE
├── nama VARCHAR(255) NOT NULL
├── alamat TEXT NOT NULL
├── user_id UUID FOREIGN KEY (references auth.users)
├── created_at TIMESTAMP (auto)
└── updated_at TIMESTAMP (auto)

RLS POLICIES
├── SELECT: Users can only view their own customers
├── INSERT: Users can only add their own customers
├── UPDATE: Users can only edit their own customers
└── DELETE: Users can only delete their own customers

INDEXES
└── customers_user_id_idx on user_id (performance)
